## About
homebrew for [python-mecab](http://mecab.googlecode.com/svn/trunk/mecab/doc/bindings.html)

## usage

### install
```
$ brew tap hhatto/python-mecab
$ brew install python-mecab
```

### uninstall
```
$ brew untap hhatto/python-mecab
$ brew uninstall python-mecab
```
